# HOWTO: Как сохранить обученную модель

После завершения цикла обучения (`model.fit`), важно сохранить состояние нейросети в файл, чтобы не обучать её заново при каждом запуске.

### Код для сохранения
Используйте расширение `.keras` (стандарт для Keras 3). Добавьте эти строки в конец вашего основного скрипта:

```python
# Сохранение архитектуры и весов в один файл
model.save('cnn_cifar10_model.keras')

# Подтверждение для консоли
print("Модель успешно сохранена в файл: cnn_cifar10_model.keras")
```
